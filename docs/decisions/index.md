# Decisions Index

## AI向け要約

- このカテゴリは、長期的に参照すべき技術判断を残す場所です。
- 一時的な作業ログではなく、なぜそうしたかを短く残します。
- 実装や運用で判断理由を確認したいときだけ参照します。
- 新規追加時はテンプレートに従い、既存判断との重複を避けます。

## このカテゴリを読むべきとき

- 過去の技術判断理由を確認したいとき
- 同種の判断を追加するとき
- 既存方針を変えるべきか検討するとき

## 運用ルール

- 一時的な作業経緯は残さない
- 変更時は既存決定を修正するか、新規決定を追加するか判断する
- 新規追加時は [docs/_templates/decision-template.md](../_templates/decision-template.md) を参照する
