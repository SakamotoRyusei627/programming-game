# Backend Index

## AI向け要約

- このカテゴリは、API、業務ロジック、データアクセス、DTO、例外処理、検証を扱います。
- ユーザー要望には Spring Boot の例が含まれていますが、現時点で採用技術は未確認です。
- サーバサイド変更時に最初にここを読み、必要な詳細だけ確認します。
- 実装方式が判明したら、具体的な層構成に更新します。

## このカテゴリを読むべきとき

- APIやサーバ処理を変更するとき
- 業務ロジックの責務を確認したいとき
- DTOやバリデーションの扱いを確認したいとき
