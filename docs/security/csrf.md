## 基本方針

- Cookie ベース認証なら CSRF を前提に考える
- SameSite だけで完結すると断定しない
- フロントとバック両方の要件を揃える

## 実装ルール

- 対策方式: 要確認

- 変更時はログインとログアウト導線も確認する
