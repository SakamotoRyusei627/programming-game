# Frontend Index

## AI向け要約

- このカテゴリは、UI実装、ルーティング、状態管理、APIクライアント、フォーム、スタイリングを扱います。
- フロントエンド実装が存在する場合の入口ですが、現時点では実装は未確認です。
- React 系の変更時に最初にここを読み、必要な詳細だけ開きます。
- 実装がない場合は、このカテゴリを読まずに進めてよい前提です。

## このカテゴリを読むべきとき

- 画面やUIを変更するとき
- ルーティングやページ遷移を変更するとき
- フロント側のAPI呼び出しや状態管理を変更するとき
