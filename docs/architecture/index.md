# Architecture Index

## AI向け要約

- このカテゴリは、システム全体像・ディレクトリ構成・外部依存を整理する入口です。
- 全体構成を把握したいときに最初に読みます。
- 実装コードが未確認のため、現時点では汎用的な確認項目を中心にしています。
- 実際の技術スタックや構成が確定したら、詳細ファイルへ具体情報を反映します。

## このカテゴリを読むべきとき

- リポジトリ全体の構造を把握したいとき
- 実装ファイルの置き場所を決めたいとき
- 外部サービスとの依存関係を確認したいとき
